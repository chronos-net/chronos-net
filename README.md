## Proyectos en desarrollo

### Sistema de Reconocimiento Facial para Control de Asistencia

Plataforma biométrica basada en inteligencia artificial para registro de asistencia mediante reconocimiento facial.

Componentes:

- enrolamiento biométrico
- generación de embeddings faciales
- verificación biométrica
- detección de duplicados
- almacenamiento vectorial

Tecnologías:

Python · FastAPI · OpenCV · InsightFace · PostgreSQL · pgvector

---

### Plataforma de Monitoreo de Infraestructura TI

Sistema de monitoreo tecnológico para supervisión de infraestructura institucional.

Capacidades:

- monitoreo de servidores
- monitoreo de VPN
- monitoreo de URLs
- monitoreo de bases de datos
- generación de alertas
- dashboard ejecutivo

### Plataforma de Control de Residencias y Servicio Social

Sistema institucional para la gestión integral de residencias profesionales y servicio social.

Funciones principales:

- registro de estudiantes
- control institucional
- seguimiento de actividades
- validación administrativa
- generación de reportes
